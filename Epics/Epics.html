Epics basics

Epics (Experimental Physics and Industrial Control System) is a bundle of code written by people in Argonne National Laboratory that helps you to build big automation projects.

EPICS is great, but my feeling is that the lack of beginners tutorials makes it difficult to learn. To help with that I've done a few tutorials about how to use epics.

These tutorials are based on personal notes in my notebook.
